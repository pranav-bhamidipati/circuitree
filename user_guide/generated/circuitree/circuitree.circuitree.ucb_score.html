circuitree.circuitree.ucb_score

circuitree.circuitree.ucb_score(graph: DiGraph, parent: Hashable, node: Hashable, exploration_constant: float | None = 1.4142135623730951, **kw)

Compute the UCB score for a node in the search graph.

The UCB score balances exploitation and exploration during tree search. It combines the mean reward of a child node with an exploration term based on the visit counts of the parent and child nodes.

Parameters:

  • graph (nx.DiGraph) – The directed search graph.

  • parent (str) – The ID of the parent node of the target node.

  • node (str) – The ID of the target node for which to compute the UCB score.

  • exploration_constant (float, optional) – The exploration constant that controls the balance between exploitation and exploration. Defaults to np.sqrt(2.0).

Returns:

The UCB score of the target node.

Return type:

float
